# MiMo TTS 调试平台

MiMo V2.5 TTS 模型的调试与测试工具，提供可视化界面用于语音合成参数配置、实时预览和 API 请求调试。

## 功能特性

### 三种合成模式

- **预置音色合成** - 内置 8 种精品音色（中文 4 种 + 英文 4 种），开箱即用
- **音色设计合成** - 通过文本描述自定义音色，支持 ASMR、旁白、电台等预设
- **音色复刻合成** - 基于音频样本精准复刻音色

### 风格控制

- **自然语言模式** - 用文字描述想要的语音风格
- **音频标签模式** - 使用标签精确控制情绪、语调、节奏
- **导演模式** - 角色、场景、指导三维度精细控制

### 开发者工具

- **API 请求预览** - 实时生成 cURL / Python / JSON 请求代码
- **一键复制** - 快速复制请求代码用于集成
- **合成历史** - 记录最近 20 条合成记录，支持回放

## 安装使用

### macOS

下载 `MiMo TTS 调试平台.app` 并拖入应用程序文件夹。

### 从源码构建

```bash
# 前置要求
# - Rust (https://rustup.rs)
# - Node.js (可选，用于开发)

cd tts-debug-app/src-tauri
cargo tauri build
```

## 使用说明

1. 点击右上角「⚙ API配置」，输入你的 MiMo API Key
2. 选择合成模式和音色
3. 输入要合成的文本
4. 点击「🎙 开始合成」
5. 合成完成后可播放预览或下载音频

## API 配置

| 参数 | 说明 |
|------|------|
| API Key | MiMo 平台 API 密钥 |
| Base URL | API 地址，默认 `https://token-plan-sgp.xiaomimimo.com/v1` |
